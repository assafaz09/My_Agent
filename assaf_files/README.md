# 📁 Assaf's Personal Files

This directory contains personal files for training Assaf's AI agent.

## 📋 Recommended Files to Upload:

### 📄 **Essential Documents**
- `resume.pdf` - Your professional resume/CV
- `bio.txt` - Personal biography and background
- `projects.docx` - Detailed project descriptions
- `skills.txt` - Technical and soft skills list

### 🖼️ **Personal Images** 
- `profile_photo.jpg` - Professional headshot
- `work_photos/` - Pictures of you working, presenting, etc.
- `life_photos/` - Personal moments that show your personality

### 🎵 **Audio Content**
- `interview.mp3` - Interviews where you talk about yourself
- `presentation.wav` - Talks or presentations you've given
- `personal_story.m4a` - You telling your story in your own words

### 📝 **Additional Content**
- `achievements.txt` - List of accomplishments
- `goals.txt` - Personal and professional goals
- `values.txt` - What's important to you in life and work
- `communication_style.txt` - How you prefer to communicate

## 🚀 How to Upload:

```bash
# Upload all files in this directory
upload_files.bat assaf_files\

# Or upload specific files
upload_files.bat assaf_files\resume.pdf assaf_files\bio.txt
```

## 📊 After Uploading:

Check your agent's knowledge:
```bash
upload_files.bat
```

## 🔒 Privacy Note:

These files are processed locally and used only to train your personal agent.
No data is shared with external services except for OpenAI API processing.
